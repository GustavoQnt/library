# Security Policy

## Reporting a Vulnerability
If you believe you have found a security vulnerability, please report it
privately.

1. Email `security@yourdomain.com` with a clear description.
2. Include steps to reproduce and the impact.
3. We will acknowledge receipt within 72 hours.

Do not open a public issue for security reports.

## Supported Versions
Security fixes are applied to the latest `main` branch.
